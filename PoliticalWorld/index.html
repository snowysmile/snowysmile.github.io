相關政治與歷史，我的理解

理論篇

1，政治是人類社會的基礎，它決定著權利的分配。

2，「群眾的眼睛是雪亮的」這句話是不折不扣的謊言。實際上真理，尤其在一開始，永遠是被少數人領悟總結。而大衆永遠是後知後覺的。

中國篇

1, 這個世界上有很多和中國相似垃圾的國家。但中國是世界上最大最惡的恐怖主義國家。謊言和暴力是這個國家的支柱；「好話說盡， 壞事做絕」是這個國家政府的牌匾。

2, 中國
真實所謂「中國」的歷史，是一群群互相敵對與仇恨的人，彼此屠殺與顛覆後，形成一個個獨立國家的故事。

秦帝國(前221-前207) → 漢帝國(前202-220) → 魏帝國(220-266) → 晉帝國(266-420) → 隋帝國(581-619) → 唐帝國(618-907) → 宋帝國(960-1279) → 元帝國(1271-1368) → 明帝國(1368-1644) → 清帝國(1636-1912) → 中華民國(1912- ) → 中國(1949- )

其彼此之間非但不是正統繼承的關係，反而常常是不共戴天的死敵。不僅有着外來民族的入侵，更往往伴隨着大規模的屠殺與人口滅亡。每每改朝換代，人口滅絕一半已經是能稱作「溫和」。

所謂「中國人」，嘴上以「同胞手足」互稱，但真實的歷史卻寫滿了吃人與屠戮：

秦國白起屠殺四十萬趙國戰俘；曹操徐州屠殺平民數十萬；唐末王仙芝焚燒殺掠江陵城約十萬戶約四十萬人；唐末黃巢廣州殺胡商十二萬至二十萬人；蒙古軍屠殺華北五百萬人、屠殺四川人三百萬；皇太極濟南屠殺約十三萬人；張獻忠屠蜀上百萬人；明末清初揚州十日八十萬、嘉定三屠五十萬；江陰、南昌各十餘萬人，留發不留頭；太平天國屠殺數十萬；曾國藩率湘軍攻破南京城屠殺數十萬。

新中國成立以來，土改鎮反屠殺處決數百萬；大饑荒變相屠殺三四千萬；十年文革造成上千萬人非正常死亡；八九六四，拋棄遮羞布直接對學生市民開槍死傷數千。

入侵與屠殺統統都可以在歷史的包裝下，被稱作為同胞手足情誼與文化的繼承與延續，作為兩角羊般的底層屁民還可以陶醉于這般吃人不吐骨頭的敘事自我感動厲害了你的國。

3，中華
1905年，中國同盟會在日本東京成立，提出「驅除韃虜，恢復中華，創立民國，平均地權」的口號。1912年，中華民國成立後，首次正式將「中華」納入國號，逐漸賦予了該詞語與民族、國家綁定的現代意義。

4，中華民族
中華民族，是一個在晚清由梁啟超發明的政治概念，其在1902年與1905年先後使用「中華民族」這一稱呼，但梁啟超在具體使用「中華民族」一詞時卻比較混亂，有時指漢族，有時又指中國的所有民族，認為「中華民族自始本非一族，實由多民族混合而成」。

5，中華文明
目前最早可考的文字是公元前1300年的商朝。考古學上最早可考為公元前1920年具有城市定位代表性的二里頭文化遺址。更久遠的歷史，包括西周文獻中的夏朝，已經全部為無考究的傳說。
